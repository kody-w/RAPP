How to Test Cross-Browser Mesh

  1. Open in Multiple Browsers: Open this page in Chrome, Firefox, Safari, and Edge simultaneously
  2. Watch Mesh Formation: All browsers will automatically discover each other via localStorage signaling
  3. Establish WebRTC Connections: Peer-to-peer data channels connect all browsers (watch the visualization!)
  4. Make Changes: Click "Add User" or "Add Chat" in ANY browser
  5. See Real-Time Sync: Watch the change propagate to ALL other browsers instantly via the mesh
  6. Test Resilience: Close one browser and see the mesh continue working with remaining nodes
  7. Check Operation Log: See distributed operations, vector clocks, and consensus
