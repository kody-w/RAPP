04The Email Draft

Send this.

Subject: 30 min demo request — a lightweight format for sharing agents across teams

I'm on the AIBAST (AI Business Applications Specialist Team) and I've been working on RAPP (Rapid Agent Prototyping Platform) — an internal effort I'd love your take on. I think it could help with a challenge that's quietly growing across Microsoft: how we share agents between teams.

What I've been noticing: Teams across the company are each building great agent tooling in parallel. The work is real and the needs are real — but agents don't always travel easily between teams, and there isn't a shared baseline for moving them around.

Why I think it matters now: In a couple of years, everyone at Microsoft is going to need a fast way to build and share agents. A shared, lightweight format on top of the great work already happening would let it compound, and it's cheaper to put that layer in place now than to retrofit it later.

To be clear: this is a complement to the excellent tooling other teams have built — not a replacement. The goal is to accelerate and make agents accessible to everyone through easier sharing and learning, using our RAPP vibe agent building tool.

What RAPP does differently:

  • One file = one agent. No frameworks, no build steps. A 14-year-old can ship their first agent in minutes.
  • Three tiers, zero modification. Same file runs locally, in Azure Functions, and in Copilot Studio on Teams.
  • Shareable by design. Agents install with a file drop. Registry, store, 7-word speakable phrases.
  • Already working. Frozen v1 spec, one-line installer, live agent store, natural-language agent generation.

My ask: 30 minutes to walk you through RAPP and get your honest feedback on whether it could complement what other teams are doing. I've attached a short video walkthrough.
