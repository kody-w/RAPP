# RAPP Development Tools

This directory contains utility scripts and tools for RAPP development and maintenance.

## Agent Catalog Generation

### generate_agent_catalog.py
Generates agent catalog files from AI-Agent-Templates metadata.

**Usage:**
```bash
# Generate manifest.json (recommended)
python tools/generate_agent_catalog.py --format manifest

# Generate JavaScript fallback
python tools/generate_agent_catalog.py --format javascript

# Generate both formats
python tools/generate_agent_catalog.py --format both
```

**Outputs:**
- `manifest.json` - Structured agent catalog (recommended)
- `agent_catalog_generated.js` - JavaScript fallback for offline use

**Documentation:** See [DATA_LOADING_PATTERN.md](/docs/guides/DATA_LOADING_PATTERN.md)

## Agent Store Testing

### test_agent_store_urls.sh
Tests agent store URL accessibility and validates links.

**Usage:**
```bash
bash tools/test_agent_store_urls.sh
```

## Generated Files

### agent_catalog_generated.js
JavaScript version of agent catalog, generated by `generate_agent_catalog.py`.
This file is auto-generated and should not be manually edited.

## Tool Development

When creating new development tools:

1. Place scripts in this `/tools` directory
2. Make scripts executable: `chmod +x tools/your_script.sh`
3. Add usage documentation to this README
4. Update main documentation if the tool affects deployment or development workflows
